# Alcance del Proyecto - Fase Inicial

## Objetivo
Crear un skeleton funcional y profesional que sirva de base para el desarrollo de los módulos de escandallo.

## Funcionalidades Incluidas (Skeleton)
1. **Infraestructura Monorepo:** Configuración base de carpetas y entornos.
2. **Autenticación Base:**
   - Registro/Login/Logout en el backend.
   - Pantallas de Login y Dashboard Protegido en el frontend.
3. **Gestión de Roles:** Sistema básico de roles (Admin, Usuario).
4. **Contratos de API:** Definición inicial de endpoints para `/api/v1/auth/*`.
5. **Layouts de Aplicación:** Skeleton de navegación y área privada.

## Fuera de Alcance (Esta Fase)
- Módulos específicos de escandallos (ingredientes, recetas, cálculos).
- Integraciones con proveedores externos.
- Sistema de archivos complejo.
- Multitenancy.
